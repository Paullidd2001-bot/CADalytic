CADalytic is aiming to be a modern, parametric 3D CAD System on a par with professional paid systems. The goal is to produce something that will be released under the GNU GPL or LGPL licence and be cross-compilable.
I have been a Draughtsman/Design Engineer for many decades and it has always pained me that (Since the mid 90's, anyhow), I haven't been able to fully escape the Microsoft world and successfully continue professional work.
Now it's time to put that right and just build the tool that I want.
